# Akash Challa - Portfolio Website

Dark-themed portfolio website with cyan accents.

## Quick Start

1. Extract the ZIP file
2. Open `index.html` in your browser
3. That's it!

## Files Included

- `index.html` - Main website
- `styles.css` - All styling
- `script.js` - Interactive features
- `Pass_photo_formal.jpg` - Your photo
- `Akash_Challa_Resume.pdf` - Your resume
- `README.md` - This file

## Features

- Dark theme with cyan accents
- Fully responsive design
- Smooth navigation
- Contact section with social links
- Project showcase
- Experience timeline
- Skills categorization

## Deployment

### Option 1: Netlify (Easiest)
1. Go to netlify.com
2. Drag the entire folder
3. Done!

### Option 2: GitHub Pages
1. Create a GitHub repository
2. Upload all files
3. Enable Pages in Settings

### Option 3: Your Domain (akashchalla.com)
1. Deploy to Netlify or Vercel
2. Add custom domain in settings
3. Update DNS records at your registrar

## Customization

To change colors, edit `styles.css`:
- `--primary-bg` - Main background color
- `--accent-color` - Cyan highlight color

## Support

If you have any issues, check:
- All files are in the same folder
- Clear browser cache (Ctrl+Shift+R)
- Try a different browser
